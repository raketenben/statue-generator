How to use:

  1. Enter username or slelect a skin
  2. Hit Generate
  3. Download Statuepack and add it to your world
  4. Use the command above to summon your statue
Pose your statues:

#multiply the angle you want by 100 --> 90Â° = 9000


#rotate the entire statue

/scoreboard players set @e[tag=statue,tag=main] statue_x 9000


#rotate the neck x

/scoreboard players set @e[tag=statue,tag=main] statue_x_head 9000


#rotate the neck y

/scoreboard players set @e[tag=statue,tag=main] statue_y_head 4500


#rotate the right arm

/scoreboard players set @e[tag=statue,tag=main] statue_x_ra 9000

/scoreboard players set @e[tag=statue,tag=main] statue_y_ra 9000


#rotate the left arm

/scoreboard players set @e[tag=statue,tag=main] statue_x_la 9000

/scoreboard players set @e[tag=statue,tag=main] statue_y_la 9000


#rotate the right leg

/scoreboard players set @e[tag=statue,tag=main] statue_x_rl 9000

/scoreboard players set @e[tag=statue,tag=main] statue_y_rl 9000


#rotate the left leg

/scoreboard players set @e[tag=statue,tag=main] statue_x_ll 9000

/scoreboard players set @e[tag=statue,tag=main] statue_y_ll 9000
