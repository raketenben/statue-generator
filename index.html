Statue generator

Create posable player-statues for Minecraft



To summon a Player-skin-statue first download the datapack with the button below.

Once you have done that copy the above commands into a mc-function and execute them.

Download Datapack

Now you can use the following commands to pose your statue:


#multiply the angle you want by 100 --> 90° = 9000 #rotate the entire statue /scoreboard players set @e[tag=imposter,tag=main] imposter_x 9000 #rotate the neck x /scoreboard players set @e[tag=imposter,tag=main] imposter_x_head 9000 #rotate the neck y /scoreboard players set @e[tag=imposter,tag=main] imposter_y_head 4500 #rotate the right arm /scoreboard players set @e[tag=imposter,tag=main] imposter_x_ra 9000 /scoreboard players set @e[tag=imposter,tag=main] imposter_y_ra 9000 #rotate the left arm /scoreboard players set @e[tag=imposter,tag=main] imposter_x_la 9000 /scoreboard players set @e[tag=imposter,tag=main] imposter_y_la 9000 #rotate the right leg /scoreboard players set @e[tag=imposter,tag=main] imposter_x_rl 9000 /scoreboard players set @e[tag=imposter,tag=main] imposter_y_rl 9000 #rotate the left leg /scoreboard players set @e[tag=imposter,tag=main] imposter_x_ll 9000 /scoreboard players set @e[tag=imposter,tag=main] imposter_y_ll 9000 #Delete the nearest statue /function imposter:kill #Disable live movement from scorebaord values #This way you can pose them with the Pose value in nbt /tag @e[sort=nearest,tag=main,limit=1] remove live

Additional Information:


This generator works best with the Steve skin, using the slim(Alex) version might cause some artifacts on the arms.
